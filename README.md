# NOLDEX LOGISTICS SARL — Site Vitrine

Marketplace B2B de location d'engins lourds en Guinée.

## Structure du projet

```
noldex-logistics/
├── index.html          # Site complet (SPA)
├── assets/
│   └── noldex_logo.svg # Logo vectoriel séparé
└── README.md
```

## Technologies

- HTML / CSS / JavaScript (Vanilla — aucun framework)
- Firebase Firestore & Storage (données engins, articles, vidéos)
- Google Fonts (Barlow Condensed, Inter, JetBrains Mono)

## Déploiement

Ce site est un fichier unique `index.html`. Il peut être déployé sur :
- **GitHub Pages** : activer dans Settings → Pages → Branch: main
- **Firebase Hosting**
- N'importe quel hébergeur statique

## Zones géographiques couvertes

Conakry · Siguiri · Boké · Kindia
