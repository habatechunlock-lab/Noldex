FAQ

Questions fréquentes

Combien coûte l'inscription sur NOLDEX LOGISTICS SARL ?+
L'inscription est gratuite pour les partenaires et les clients B2B. NOLDEX LOGISTICS SARL perçoit une commission sur les locations effectuées via la plateforme.
Quels types d'engins puis-je louer ?+
Camions benne, chargeuses, pelles mécaniques, grues, citernes, compacteurs et autres engins lourds. La liste s'agrandit avec les partenaires qui rejoignent la plateforme.
Dans quelles régions de Guinée êtes-vous disponibles ?+
Nous couvrons actuellement Conakry, Boké, Siguiri et Kindia. D'autres régions seront ajoutées progressivement selon la demande des partenaires.
Comment fonctionne le suivi GPS ?+
Chaque engin inscrit sur la plateforme est équipé d'un tracker GPS (Traccar ou Wialon). Le client et le partenaire peuvent suivre la localisation en temps réel depuis leur tableau de bord.
Les contrats digitaux sont-ils juridiquement valables en Guinée ?+
Oui. Les contrats de location (CCL) émis par NOLDEX LOGISTICS SARL sont conformes au droit guinéen et intègrent une signature électronique reconnue pour les deux parties.
Quels sont les délais pour obtenir un devis ?+
NOLDEX LOGISTICS SARL s'engage à répondre à toutes les demandes de devis sous 24 heures ouvrées. Vous recevez une confirmation par SMS et par email.

FAQ Tarifs

Questions sur la tarification

Comment fonctionne la commission NOLDEX ?+
NOLDEX perçoit une commission de 8 à 12% sur chaque location réalisée via la plateforme. Cette commission est déduite automatiquement au moment du paiement. Elle couvre les coûts de la plateforme, le suivi GPS, la gestion des contrats et le support.
Y a-t-il des frais d'inscription ou d'abonnement ?+
Non. L'inscription est entièrement gratuite pour les clients et les partenaires. Vous ne payez aucun frais fixe mensuel. NOLDEX se rémunère uniquement sur les transactions réussies.
Les tarifs affichés incluent-ils le carburant ?+
Les tarifs de location n'incluent pas le carburant, sauf mention contraire dans le devis. Le carburant est à la charge du client pour la plupart des engins, sauf arrangement particulier avec le partenaire propriétaire.
Puis-je négocier le tarif pour une longue durée ?+
Oui. Les locations supérieures à 2 semaines donnent lieu à des tarifs préférentiels. Pour les contrats mensuels ou les besoins récurrents, des prix dégressifs sont systématiquement proposés dans le devis.
Quels moyens de paiement sont acceptés ?+
Orange Money, MTN Mobile Money et virement bancaire. Nous travaillons à l'intégration de Wave et d'autres solutions de paiement mobile guinéennes.

📋 Script SQL — Créer les tables Supabase 
-- Copiez ce script dans Supabase → SQL Editor → Run
create table if not exists engins      (id text primary key, nom text, type text, proprio text, ville text, cap text, prix numeric, statut text, gps boolean, emoji text, validated boolean, created_at timestamptz default now());
create table if not exists clients     (id text primary key, societe text, contact text, tel text, email text, secteur text, ville text, locations int default 0, total numeric default 0, statut text, created_at timestamptz default now());
create table if not exists contrats    (id text primary key, client text, "enginId" text, engin text, chantier text, debut text, fin text, montant numeric, statut text, created_at timestamptz default now());
create table if not exists demandes    (id text primary key, client text, engin text, chantier text, debut text, fin text, statut text, created_at timestamptz default now());
create table if not exists paiements   (id text primary key, "contratId" text, client text, montant numeric, date text, mode text, statut text, created_at timestamptz default now());
create table if not exists maintenance (id text primary key, "enginId" text, engin text, type text, description text, date text, cout numeric, statut text, created_at timestamptz default now());
create table if not exists partenaires (id text primary key, nom text, contact text, tel text, email text, ville text, engins int default 0, created_at timestamptz default now());
create table if not exists platform_params (id text primary key default 'main', nom text, email text, tel text, adresse text, commission numeric, "commPart" numeric, tva numeric, devise text, om text, mtn text, rib text, rccm text, nif text, site text);
